#gieba

jieba in golang

jieba url: https://github.com/fxsjy/jieba

##install

```bash
$ go get github.com/pastebt/gieba
```

##setup data
```bash
$ mkdir /usr/share/gieba
$ cd /usr/share/gieba
$ ln -s $GOROOT/src/pkg/github.com/pastebt/gieba/data/ .
```

##test
```bash
$ go test github.com/pastebt/gieba
```
